# iFace 1.0 Roadmap

当前版本：`1.4.2`

iFace 1.0.0 的目标不是堆更多页面，而是把现有刷题、笔记、AI 反馈、数据备份这几条核心链路打磨到稳定可日用。

## 1.0 定义

1. 打开即用：不配置 AI 也能完整刷题、记录进度、复盘薄弱点。
2. 数据可信：题库、学习记录、笔记、AI 复盘都能稳定保存、导出、恢复、同步。
3. 刷题顺手：三种答题模式、键盘快捷键、语音输入、笔记抽屉、练习会话都没有明显打断感。
4. AI 可控：模型配置清晰，默认 Prompt 稳定，AI 反馈能沉淀为笔记，而不是只停留在一次性对话。
5. 视觉稳定：桌面和移动端布局不抖动、不挤压、不出现明显可访问性问题。

## 版本阶段

### 0.14.x：练习主流程稳定

- 固化三种答题模式：先答题、边看边记、纯记忆。
- 收尾快捷键：`Space`、`1/2/3`、方向键、`N`、`A`、`Esc` 的冲突处理。
- 完善语音输入：浏览器支持检测、识别中状态、错误提示、停止逻辑。
- 优化练习会话：大题单 session 存储、结束总结、重练未掌握题。
- 补齐桌面布局稳定性：滚动条槽位、导航 tab 尺寸、按钮文字溢出。

### 0.15.x：笔记与复盘系统

- 题目笔记作为独立复盘入口：抽屉、快捷键、自动保存、Markdown 预览。
- 题库列表支持只看有笔记、搜索笔记内容、最近笔记排序。
- Dashboard 增加最近笔记与下一步学习建议。
- AI 作答反馈一键保存为复盘笔记。
- 后续考虑：笔记模板、笔记标签、按笔记更新时间生成复习队列。

### 0.16.x：AI 面试教练成熟化

- 模型服务商预设保持更新：OpenAI、DeepSeek、Qwen、智谱、自定义兼容接口。
- 默认 Prompt 分层：AI 对话、作答批改、快捷动作、保存为笔记各自有明确目标。
- 设置页增加配置可理解性：模型说明、Base URL 校验、API Key 本地存储说明。
- AI 回答体验优化：流式跟随、手动滚动暂停、错误恢复、清空会话、复制与重试。
- 后续考虑：按题目模块自动调整 AI 反馈重点。

### 0.17.x：数据安全与同步

- 导入 / 导出覆盖题库、学习记录、题目笔记、AI 会话关键数据。
- Gist 同步合并策略稳定：本地与云端按更新时间合并，避免旧数据覆盖新笔记。
- 设置页展示数据统计与备份状态。
- 增加数据恢复前的风险提示和结果反馈。
- 1.0 前必须完成：从旧备份版本恢复的兼容性检查。

### 0.18.x：题库质量与内容工程

- `check:questions` 作为发版前必跑检查。
- 题目 schema 校验、ID 重复检查、难度分布、模块覆盖统计、内置题库 registry 和默认分类一致性。
- 持续优化前端、Go、AI Agent 题库内容质量。
- 统一题目答案风格：面试口述版、关键点、易错点、示例代码。
- 后续考虑：题目质量评分和低质量题目待修清单。

### 0.19.x：PWA、可访问性与发布体验

- PWA 安装体验、离线缓存、更新提示打磨。
- 移动端关键页面体验复查：题目详情、笔记抽屉、设置页、导入页。
- 可访问性：按钮 aria、焦点样式、键盘操作、颜色对比。
- README、截图、功能说明更新到最新状态。
- 建立 1.0 发版前人工验收清单。

### 1.0.0：稳定版

1. `bun run check:all` 通过，覆盖代码规范、题库质量、Gist 同步兼容和生产构建。
   - 其中 `check:version` 覆盖 `package.json`、README 徽章、Roadmap、Smoke 记录和设置页版本展示。
   - 其中 `check:release` 在版本切到 `1.0.0` 后要求发版审计无未完成项、外部服务 smoke 有通过证据，且发布说明不再是草稿。
   - 其中 `check:quality-gate` 防止完整门禁、发版脚本或文档门禁列表漏掉关键子检查。
   - 其中 `check:docs` 覆盖 README/docs 本地链接、License 和截图资源。
   - 其中 `check:backup` 覆盖本地备份导入解析、旧备份兼容推导和错误输入拒绝。
   - 其中 `check:ai` 额外覆盖模型预设、AI Prompt、作答反馈上下文和流式解析。
   - 其中 `check:pwa` 在生产构建后覆盖 manifest、Service Worker、Workbox、图标资源和内置题库 JSON 预缓存。
2. 核心路径人工验证通过：
   - 导入内置题库
   - 题库筛选、搜索、按笔记筛选
   - 三种答题模式
   - 题目笔记新增、编辑、搜索、最近排序
   - AI 配置、AI 对话、作答反馈、保存为笔记
   - 薄弱点练习、专项练习、练习会话结束总结
   - 数据导出、导入、Gist 同步
3. 无已知数据丢失风险。
4. 桌面和移动端主要页面无明显布局错位。
5. README、截图、版本号、Release notes 与实际功能一致。

## 近期优先级

1. 继续做练习页和题目详情页的真实使用细节：快捷键冲突、空状态、会话结束、语音输入边界。
2. 打磨数据安全：备份恢复前预览、同步冲突提示、设置页数据健康状态。
3. 做 1.0 验收清单和手动 smoke test 流程，避免后续改动破坏主流程。
4. 更新文档和截图，让 README 反映当前版本真实能力。
